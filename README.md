# 📊 HR Employee Attrition Analysis using Microsoft Excel

## 📌 Project Overview

This project focuses on analyzing **HR Employee Attrition** using Microsoft Excel. The objective is to identify the factors influencing employee attrition and provide insights that help organizations improve employee retention and workforce management.

The project includes data cleaning, preprocessing, pivot table analysis, interactive dashboards, and visualizations to transform raw HR data into meaningful business insights.

---

## 🎯 Objectives

- Analyze employee attrition patterns.
- Identify factors affecting employee turnover.
- Compare attrition across departments and job roles.
- Analyze employee demographics.
- Study the impact of overtime, salary hike, experience, and work-life balance on attrition.
- Create an interactive dashboard for HR decision-making.
